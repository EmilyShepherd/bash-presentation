What is a Shell?

  • A shell is a user's entry point into a computer
  • It "wraps" up the system's functionality
  • Typically come in two forms:
    • Command Line Interpreter (CLI)
    • Graphical User Interface (GUI)

What is the CLI

  • "Interprets" user input, to perform a command
  • Has a very simplistic, syntax
    • Primary function is just to run programs on the machine
    • Very limited logical / conditional operations
    • Scripting is not programming

Bash, the Bourne Again Shell

  • One of the earliest shells, sh, was developed by Stephen Bourne back in 1977
  • Bourne Again Shell (BASH), a successor to sh, released in 1989
  • Systems almost always have sh and bash still on them, or some form of backwards compatible replacement of them
